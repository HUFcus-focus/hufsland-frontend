## Introduction

패키지 매니저로 `yarn`을 사용합니다. 없는 경우 다음 명령어로 설치해주세요

```
npm install -g yarn
```

다음 명령어로 프로젝트의 `dependencies`를 설치해주세요

```
yarn
```

다음 명령어로 `http://localhost:3000`에 개발 환경을 실행할 수 있습니다

```
yarn dev
```

> 코드 컨벤션을 위해 `ESLint`와 `Prettier`, 커밋 메시지 컨벤션을 위해 `gitmoji-cli`를 적용해주세요!
